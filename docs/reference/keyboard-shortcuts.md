# Keyboard Shortcuts

Complete list of keyboard shortcuts in TaleNode.

!!! note
    On macOS, ++ctrl++ corresponds to ++cmd++.

## File Operations

| Shortcut | Action |
|---|---|
| ++ctrl+n++ | New project |
| ++ctrl+o++ | Open project |
| ++ctrl+s++ | Save project |

## Edit Operations

| Shortcut | Action |
|---|---|
| ++ctrl+z++ | Undo |
| ++ctrl+shift+z++ | Redo |
| ++ctrl+a++ | Select all nodes |
| ++ctrl+d++ | Duplicate selected nodes |
| ++delete++ / ++backspace++ | Delete selected nodes |

## Navigation

| Shortcut | Action |
|---|---|
| Middle mouse drag | Pan canvas |
| ++space++ + left mouse drag | Pan canvas (alternative) |
| Scroll wheel | Pan canvas |
| ++ctrl++ + scroll wheel | Zoom in/out |
| Trackpad pinch (macOS) | Zoom in/out |
| ++f++ | Zoom to fit (show all nodes) |
| ++escape++ | Close search bar / Exit sub-graph |

## Search & Replace

| Shortcut | Action |
|---|---|
| ++ctrl+f++ | Open search bar |
| ++ctrl+h++ / ++cmd+shift+h++ | Open search bar with replace |
| ++escape++ | Close search bar |
| ++enter++ | Next search result |

## Canvas Interaction

| Action | Input |
|---|---|
| Select node | Left-click on node |
| Deselect all | Left-click on empty canvas |
| Box select | Left-click drag on empty canvas |
| Move nodes | Left-click drag on selected node(s) |
| Create wire | Left-click drag from port to port |
| Context menu | Right-click on canvas |
| Click minimap | Navigate to that area of the graph |
| Enter sub-graph | Double-click on SubGraph node |
| Node tooltip | Hover over node (shows content preview) |
